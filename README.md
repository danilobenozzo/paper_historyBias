# paper_historyBias
This repository contains scripts and resources to reproduce Figures 4 and 6 from the paper.

Contents

env_file.txt — specifies the dependencies to create the required Conda environment.

figXyy.py — scripts to reproduce the panels yy of figure X.

figures/ — folder containing the plotted figures generated by the scripts.

pvals/ — folder containing .txt files with the corresponding (time-bin) p-values for each figure.
